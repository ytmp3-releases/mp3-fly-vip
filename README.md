## Installation

**Requirements:** Windows 10 / 11 64-bit

1. Download `mp3-fly_1.1.0_x64-setup.exe`
2. Run the installer
3. Follow the instructions

> **Windows SmartScreen showing a warning?** This is expected for software without a digital signing certificate. Click **More info** → **Run anyway**. The software contains no viruses, no spyware, no toolbars.

---

## Usage

1. Paste a YouTube URL (video or playlist)
2. Choose the format — **MP3** or **MP4**
3. Select the quality
4. Choose the destination folder
5. Click **Download**

Files are automatically named. Playlists are organized into subfolders. MP3 files include the YouTube thumbnail as album artwork.
